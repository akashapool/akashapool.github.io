The Gradients

74 gradients to choose from, hand-picked from the uiGradients collection.

Here's a list of all the gradients and their class names. To change a section's gradient, replace its last class with the gradient class of your choice -
ie. <div class="col-md-12 section aubergine" id="section2">

Hover your mouse over each gradient to see what it looks like inverted. Want to use the inverted gradient? Simply add -inverse to the end of the gradient class name,
eg. aubergine becomes aubergine-inverse.

The Theme

Make it your own! Be sure to read the README.md file to get started.

74 Gradients

Choose from 74 different gradients. Want to flip it? Add -inverse to the end of its class name.

For example, change purple-bliss to purple-bliss-inverse.
